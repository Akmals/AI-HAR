# AI-HAR
Build a machine learning model that classifies which of 18 human activities a person is performing based on smartphone accelerometer and gyroscope sensor data from 3-second windows. DATA SET- Human Activity Recognition (HAR) https://www.kaggle.com/datasets/niloy333/kuhar?utm_source=chatgpt.com

Rubric-Tasks/Requirements
1. Dataset Exploration
2. Annotated Signal Exploration
3. Signal Preprocesesing
4. Windowing Strategies
5. Feature Extraction and Analysis
6. Modeling
7. Advanced Modeling and Evaluation
8. Literature Review
9. Project Report
